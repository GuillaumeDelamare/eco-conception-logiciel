    Run the demo
    To run the demo of this bundle, enter the following commands in the OSCAR shell

    obr -u http://oscar-osgi.sf.net/br/bundlerepository.txt http://www-adele.imag.fr/~donsez/dev/osgi/obr.txt
obr -s Hello Service Impl 1.0 en
    or 
    start http://www-adele.imag.fr/~donsez/dev/osgi/helloservice10/helloservice10.jar
    or use this OSCAR shell script http://www-adele.imag.fr/~donsez/dev/osgi/script/hello.txt
